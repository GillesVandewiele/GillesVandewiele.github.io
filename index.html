Solving class imbalance by re-sampling

  • 04 October 2016
    • •
  • Machine learning

When dealing with a classification problem, an imbalance in the distribution of classes can cause the model to be heavily biased towards to the class having the largest amount of samples. In order to overcome this, re-sampling techniques can be applied to balance the dataset. This article discusses the balanced accuracy metric, oversampling and undersampling. Moreover, 9 different techniques (using the imbalanced-learn library in python) are compared by presenting measurements for accuracy, balanced accuracy, model complexity, and computational complexity on 17 publicly available datasets from the UCI Machine Learning Repository.
